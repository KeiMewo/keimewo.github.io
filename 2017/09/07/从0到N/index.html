从0到N

发表于 2017-09-07 | 分类于 阅读

《失控：机器、社会与经济的新生物学》一书，通过解析现有的技术变革、社会组织形态与经济运行中蕴含的生物逻辑，对其未来发展进行展望。

生物逻辑，是生命的诞生、成长、繁衍与多样化，是从无到有、从简单到复杂、从可控到失控、从有序到稳定的非稳态的过程，即从0到N。生物是大自然的创造，生物逻辑则是大自然的馈赠——人类可以像自然学习，从无创造有、构建复杂性。

作者在书中最后一章总结了无中生有的造物九律。九律均为组织原则，适用于作者在全书探讨的机器、社会和经济的运作等宏观层面，同样地对于个体和企业组织的行动管理等微观层面具有深刻的启迪意义。

  • 分布式状态

从个体到组织，从意识到行为，无一不能被分布到众多更小的单元上。分布式状态的特征是去中心化，网络化，和多维度并行。

于个体，我们的生命形态本身即是分布式状态。大脑并非中心控制，而是诸多小单元各司其职、发挥功能。理解这种分布式状态，将指导我们更好地挖掘和锻炼自己的特长——仅仅需要局部而非整体的提升，将某一小单元的能力最大化，同时分布式状态之间的交互作用可能带来惊喜。

于组织，参照生物的分布式状态将带来进化壮大：一方面，在组织架构上，去中心化，提高决策效率，更好地发挥协同效应；一方面，在业务发展上，多线并行，网络式铺开战略布局，构筑复杂的有机生态。

  • 自下而上的控制

在分布式状态要求去中心化时，自上而下的中央集权控制也将分崩离析，取而代之的是网络模式自下而上的控制。自下而上的控制意味着反馈和适应，从行动中获得反馈，进而思考，再有下一步的行动，快速地变化和适应。

我们时常陷入一种“自上而下”控制的困局——过度思虑而迟迟不能行动。想要有所得，必须先行动，从最基础做起，由行动带动思考，再由思考引导行动。

组织要想实现全面的控制，其实反而需要先“失控”——提供集体智慧发挥的空间，当底层的基本单位相互连接且以并行方式开展行动，一切都会自然而然地发生，产生无秩序的有序。

  • 培养递增收益

资源的占有方，将通过对资源的配置吸引获取更多的资源。个人的优势特长，会在不断地利用展示过程中得到强化巩固，企业亦然。企业的品牌优势、客户群优势，在良好地运营下会带来滚雪球效应——这也可以解释在当下的市场竞争中，企业们努力在做的就是树立品牌、争抢客群。

  • 模块化生长

生物进化的逻辑是从简单到复杂，一个复杂的系统要由简单的模块组装而成。个人的成长，需要学识经验的积累，由对单一的模块的认知逐步搭建成有体系的认知系统，再将知识转化为实践的能力。而组织的成长，也是由简入繁，一步步做加法。模块化生长的要义在于，复杂性的培养需要时间，罗马非一日建成。同时，简单模块的组装，次序也很重要，模块之间应该相辅相成。个人知识和经验的不匹配，企业经营和管理的不匹配，都难以实现进步。

  • 边缘最大化

关注边缘，异变往往出自边缘。

就个体而言，边缘可能意味着思维中的盲点。通过找到盲点并加以攻克，个人的创造力能得到更好的释放。

就组织而言，在分布式状态下，各组成单元间相对独立，对边缘的忽视并不会导致组织的衰亡。但关注边缘的意义在于使得各个单元乃至整体更具有活力，富有弹性和适应力，同时边缘往往是创新的源泉。

  • 礼待错误

凯文·凯利在书中指出，进化可以被看作是对系统错误的管理。进化是一种学习，从错误中学习。

个体和组织的行为，其本质都可以视为试错行为。宽容错误，再从错误中反思总结，是通往正确的必经之路。一个成功的企业，理应有包容的文化——不可能永远只做正确的事，而要知道如何做正确的事，恰恰这种“知道”是必须从错误中得来的教训。

  • 不求目标最优，但求目标众多

多元化和纵向深入其实并不矛盾，因为评价指标绝非单一的。既然标准是多指向的，目标也没有最优，同样是多指向的。

个体以全才或专才为目标没有优劣之分。即使是选择成为专才，同样有多维的评价角度，而非唯一的精准衡量。

组织的扩张，是一个复杂化的过程，无论多元化还是垂直发展都应该有多个目标。复杂化可能带来效率的降低（相较于简单组织），但也带来了生命力和稳定性等方面的优化。

  • 谋求持久的不均衡

不均衡（不停的变化）是危机四伏，均衡（恒久不变）意味着固步自封。在不均衡与均衡之间把握一个度，即持久的不均衡，不坠落也不停止，保持持续的发展。这意味着时常更新，更新目标，从内部和外界获得反馈，主动迎接挑战和变化。

  • 变自生变

谋求持久的不均衡，需要变化，而在分布式状态、自下而上的控制下，变化将催生变化。底层的变化相互连接，触发整体的变化，而整体的变化，又使得那些带来底层变化的规则发生变化。个体和组织的进化，最终需要的是自我变化的能力。不单纯依赖外界的刺激，而是内部驱动自身改变，方能应对万变，立于不败之地。

自我变化的能力如何获取？这需要时间的积淀，是在与外界互动的过程中学习。个人以为，那些存在了百年以上的企业，可以被认为是掌握了自我变化的规则，而非被动地改变以应对外界的挑战。

作者认为，对造物九律的坚守将大获成功。但作者未能阐明的是，无中生有的时间消耗和艰巨程度。大自然孕育生命用了数十亿年，机器、社会和经济借鉴自然生物逻辑必将面临诸多挑战，路漫漫其修远兮。而对于个体和企业组织，从0到N的启发实际是从1到N，是对成长进化的认知和管理，是使个人最大化自我价值，使企业成就伟大梦想。

后记：一篇敷衍的八股式“书评”，不嫌丢人现眼地贴出来，就当给自己的警示。书其实没有太读懂，还需要反复咀嚼，有了新感想再更。
